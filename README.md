# Ev Fiyatları Tahmini (House Price Prediction)

## Amaç
Bu proje, **Yapay Zeka Temelleri Vize-Final Ödevi** kapsamında hazırlanmıştır. Amaç, Kaggle üzerinde bulunan ev fiyatları veri seti kullanılarak farklı makine öğrenmesi algoritmaları ile ev fiyatlarını tahmin etmektir.

## Veri Seti
- Kaynak: [Kaggle House Prices Dataset](https://www.kaggle.com/c/house-prices-advanced-regression-techniques)
- İçerik: Evlerin özellikleri (oda sayısı, metrekare, konum vb.) ve satış fiyatları.

## Kullanılan Algoritmalar
